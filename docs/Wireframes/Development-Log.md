# Development Log

This document tracks the ongoing development progress of the *Building AI with Simon* website. It serves as a chronological record of features built, decisions made, issues encountered, and improvements implemented.

---

## 1. Purpose of This Log

- Track all development activity over time
- Document technical decisions and reasoning
- Record challenges and solutions
- Provide transparency in the learning process
- Support consistent project growth

---

## 2. Development Approach

This project follows a **learning-first, build-in-public approach**:

- Small iterative improvements
- Continuous documentation
- Simple static architecture (HTML, CSS, minimal JS)
- Focus on clarity over complexity

---

## 3. Log Format

Each entry follows this structure:

- Date
- Feature / Task
- What was done
- Tools used
- Issues encountered (if any)
- Resolution (if applicable)
- Notes / Learnings

---
